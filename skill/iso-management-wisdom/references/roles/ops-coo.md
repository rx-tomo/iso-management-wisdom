# 経営企画/事業責任者（COO/経営企画）（Role Skill）

## このロールの専門性
- 戦略→KPI→実行計画へ落とし込める
- 横断調整（部門間・外部委託・利害関係者）
- ポートフォリオ視点（優先度/中止判断）

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 37000:2021
- 参照: `https://www.iso.org/standard/65036.html`
- 役割責任（監督/執行/牽制）と意思決定ルールを定義する

### ISO 31000:2018
- 参照: `https://www.iso.org/standard/65694.html`
- リスク基準（影響/確率尺度・受容基準）を定義する
- リスク登録簿を運用し、定期レビューとエスカレーションを行う

### ISO 37301:2021
- 参照: `https://www.iso.org/standard/75080.html`
- 教育・コミュニケーション・通報の仕組みを整備する

### ISO 19011:2018
- 参照: `https://www.iso.org/standard/70017.html`
- 指摘→是正→フォローアップを改善バックログへ接続する

### ISO 9001:2015
- 参照: `https://www.iso.org/standard/62085.html`
- 顧客要求と適用範囲（スコープ）を定義する
- プロセスとKPIを定義し、運用統制（変更管理含む）を行う

### ISO 45001:2018
- 参照: `https://www.iso.org/standard/63787.html`
- 危険源の特定とリスク評価、管理策を運用する
- 労働者参加・教育・コミュニケーションを整備する

### ISO 45003:2021
- 参照: `https://www.iso.org/standard/64283.html`
- 心理社会的リスクを特定し、対策と効果測定を回す

### ISO/IEC 38500:2024
- 参照: `https://www.iso.org/standard/81684.html`
- ITが価値/リスク/適合に与える影響をレビューする

### ISO/IEC 42001:2023
- 参照: `https://www.iso.org/standard/42001`
- AI利用の方針・目的・責任分界を定義し、経営として承認する

### ISO/IEC 20000-1:2018
- 参照: `https://www.iso.org/standard/70636.html`
- サービスカタログとSLA、役割責任を定義する

### ISO 21502:2020
- 参照: `https://www.iso.org/standard/74947.html`
- 目的・成果物・制約を明確化し、計画を策定する

### ISO 21503:2022
- 参照: `https://www.iso.org/standard/82868.html`
- 便益（ベネフィット）と依存関係を管理する

### ISO 10002:2018
- 参照: `https://www.iso.org/standard/71580.html`
- 苦情のエスカレーション基準と顧客へのフィードバックを整備する

### ISO 30414:2025
- 参照: `https://www.iso.org/standard/30414`
- データ要件・品質管理・開示（内部/外部）を運用する

### ISO 30415:2021
- 参照: `https://www.iso.org/standard/71164.html`
- D&I方針・目標・施策・測定の運用を整備する

### ISO 26000:2010
- 参照: `https://www.iso.org/standard/42546.html`
- ステークホルダー関与と説明責任（開示/対話）を運用する

### ISO 14064-1:2018
- 参照: `https://www.iso.org/standard/66453.html`
- 組織のGHG算定境界・算定ルール・証跡・報告を定義する

### ISO 14001:2015
- 参照: `https://www.iso.org/standard/60857.html`
- 目的・計画・運用管理（変更管理含む）を整備し、実行する

### ISO 22301:2019
- 参照: `https://www.iso.org/standard/75106.html`
- 重要業務の特定（BIA）と継続戦略（RTO/RPO等）を策定する

### ISO 50001:2018
- 参照: `https://www.iso.org/standard/69426.html`
- エネルギーレビューを実施し、指標（EnPI）とベースラインを定義する

### ISO 55001:2024
- 参照: `https://www.iso.org/standard/83054.html`
- 資産台帳とライフサイクル計画（更新/廃棄/保全）を整備する

### ISO 56001:2024
- 参照: `https://www.iso.org/standard/79278.html`
- 探索→検証→実装のポートフォリオ運用と学習（知識化）を回す

### ISO 30401:2018
- 参照: `https://www.iso.org/standard/68683.html`
- 知識の特定・作成・共有・維持のプロセスと責任を定義する
- ナレッジの更新・棚卸し・再利用の運用（文書化情報）を回す

### ISO 30409:2016
- 参照: `https://www.iso.org/standard/64150.html`
- 要員需要と供給を見積もり、ギャップ施策（採用/育成/外注等）を計画する

### ISO 22316:2017
- 参照: `https://www.iso.org/standard/50053.html`
- 組織レジリエンスの自己評価と強化施策のレビューを運用する

### ISO 23592:2021
- 参照: `https://www.iso.org/standard/76358.html`
- サービスエクセレンスの原則とモデルに基づき、現状診断と改善を回す

### ISO 9004:2018
- 参照: `https://www.iso.org/standard/70397.html`
- 組織の持続的成功に向け、成熟度自己評価と改善を運用する

### ISO 37101:2016
- 参照: `https://www.iso.org/standard/61885.html`
- サステナビリティの方針・目標・運用・評価・改善をMSとして整備する

### ISO/TS 14092:2020
- 参照: `https://www.iso.org/standard/68509.html`
- 地域/コミュニティの適応計画の要求と手順を整備する

### ISO 14093:2022
- 参照: `https://www.iso.org/standard/68511.html`
- 適応資金（助成等）の要求と評価の枠組みを整備する

### ISO 14097:2021
- 参照: `https://www.iso.org/standard/72433.html`
- 投融資活動に関する気候関連の評価と報告の枠組みを整備する
