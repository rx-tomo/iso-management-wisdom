# 内部監査/内部レビュー責任者（Role Skill）

## このロールの専門性
- 監査（計画/実施/報告/是正フォロー）の技法
- 独立性・客観性・証拠ベース
- 指摘を改善バックログへ接続できる

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 37000:2021
- 参照: `https://www.iso.org/standard/65036.html`
- ガバナンスの有効性を定期レビューし、改善を指示する

### ISO 37301:2021
- 参照: `https://www.iso.org/standard/75080.html`
- 監視・測定により実効性を評価し、是正・改善する

### ISO 19011:2018
- 参照: `https://www.iso.org/standard/70017.html`
- 監査プログラム（計画/頻度/範囲）を策定する
- 監査の実施（証拠収集、インタビュー、記録）を行う
- 指摘→是正→フォローアップを改善バックログへ接続する

### ISO 9001:2015
- 参照: `https://www.iso.org/standard/62085.html`
- 不適合の是正（CAPA）と継続的改善を運用する

### ISO 45001:2018
- 参照: `https://www.iso.org/standard/63787.html`
- 事故/不適合の報告・調査・是正を運用する

### ISO/IEC 27001:2022
- 参照: `https://www.iso.org/standard/82875.html`
- 内部レビューとマネジメントレビューを実施する

### ISO/IEC 20000-1:2018
- 参照: `https://www.iso.org/standard/70636.html`
- 継続改善（KPI、レビュー、是正）を回す

### ISO 20400:2017
- 参照: `https://www.iso.org/standard/63026.html`
- サプライヤの評価・改善（監査/是正）を運用する

### ISO 14001:2015
- 参照: `https://www.iso.org/standard/60857.html`
- 監視測定・内部レビュー・是正により継続改善する

### ISO 22301:2019
- 参照: `https://www.iso.org/standard/75106.html`
- 事象対応のレビューと是正・改善を運用する

### ISO 50001:2018
- 参照: `https://www.iso.org/standard/69426.html`
- 計測データ要件と改善計画を整備し、効果測定を回す

### ISO 30401:2018
- 参照: `https://www.iso.org/standard/68683.html`
- ナレッジの更新・棚卸し・再利用の運用（文書化情報）を回す

### ISO 37002:2021
- 参照: `https://www.iso.org/standard/65035.html`
- 受付→トリアージ→調査→是正→再発防止の記録とレビューを行う

### ISO 37003:2025
- 参照: `https://www.iso.org/standard/37003`
- 不正インシデントの調査、是正、再発防止を運用する

### ISO 37302:2025
- 参照: `https://www.iso.org/standard/37302`
- コンプライアンス施策の有効性評価（KPI/監視測定/調査）を設計する

### ISO 14064-3:2019
- 参照: `https://www.iso.org/standard/66455.html`
- GHG声明の検証/妥当性確認に備え、証跡と是正プロセスを整備する
