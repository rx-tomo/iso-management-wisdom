# コンプライアンス責任者（CMS）（Role Skill）

## このロールの専門性
- 順守義務の特定/更新（法規・契約・社内規程）
- 教育・通報・是正の運用設計
- 独立性と実効性評価（モニタリング）

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 37301:2021
- 参照: `https://www.iso.org/standard/75080.html`
- 順守義務（法規・規程・契約）の特定と更新を行う
- 教育・コミュニケーション・通報の仕組みを整備する
- 監視・測定により実効性を評価し、是正・改善する

### ISO 37001:2025
- 参照: `https://www.iso.org/standard/37001`
- 贈収賄防止方針と責任者体制を定義する
- 贈収賄リスク評価とリスク低減策（統制）を設計する
- 第三者デューデリジェンスと取引管理（贈答/接待等）を運用する
- 通報・調査・是正・再発防止を運用する

### ISO 37002:2021
- 参照: `https://www.iso.org/standard/65035.html`
- 通報チャネル、守秘、報復防止を含む内部通報の運用を整備する
- 受付→トリアージ→調査→是正→再発防止の記録とレビューを行う

### ISO 37003:2025
- 参照: `https://www.iso.org/standard/37003`
- 不正リスク評価と統制設計（予防/検知/対応）を整備する

### ISO 37302:2025
- 参照: `https://www.iso.org/standard/37302`
- コンプライアンス施策の有効性評価（KPI/監視測定/調査）を設計する

### ISO 37303:2025
- 参照: `https://www.iso.org/standard/37303`
- コンプライアンスに必要な力量（役割別スキル）を定義し、教育と評価を運用する
