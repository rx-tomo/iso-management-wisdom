# カスタマーサポート責任者（CS）（Role Skill）

## このロールの専門性
- 苦情/問い合わせの運用設計（SLA/分類/エスカレ）
- 顧客対応の品質管理（教育/モニタリング）
- VOCを改善へ繋げる（原因分析・施策化）

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 10002:2018
- 参照: `https://www.iso.org/standard/71580.html`
- 苦情受付・分類・原因分析・是正・再発防止の運用を整備する
- 苦情のエスカレーション基準と顧客へのフィードバックを整備する

### ISO 10004:2018
- 参照: `https://www.iso.org/standard/71582.html`
- 顧客満足の指標設計と測定・レビューを行う

### ISO 10001:2018
- 参照: `https://www.iso.org/standard/71579.html`
- 顧客対応の行動規範（約束）を策定し、周知・逸脱対応を整備する

### ISO 10003:2018
- 参照: `https://www.iso.org/standard/71581.html`
- 外部紛争解決（ADR等）へのエスカレーション基準と記録を整備する

### ISO 18295-1:2017
- 参照: `https://www.iso.org/standard/64739.html`
- コンタクトセンターの運用要件（応対品質/教育/モニタリング）を整備する

### ISO 18295-2:2017
- 参照: `https://www.iso.org/standard/64740.html`
- 委託先コンタクトセンターの要求・評価・責任分界（SLA等）を定義する

### ISO 23592:2021
- 参照: `https://www.iso.org/standard/76358.html`
- サービスエクセレンスの原則とモデルに基づき、現状診断と改善を回す
