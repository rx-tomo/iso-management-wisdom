# 品質管理責任者（QMS）（Role Skill）

## このロールの専門性
- プロセス設計（品質KPI/是正/再発防止）
- 顧客要求と変更管理の統合
- 継続改善（CAPA、内部レビュー）

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 19011:2018
- 参照: `https://www.iso.org/standard/70017.html`
- 監査プログラム（計画/頻度/範囲）を策定する

### ISO 9001:2015
- 参照: `https://www.iso.org/standard/62085.html`
- 顧客要求と適用範囲（スコープ）を定義する
- プロセスとKPIを定義し、運用統制（変更管理含む）を行う
- 不適合の是正（CAPA）と継続的改善を運用する

### ISO 21502:2020
- 参照: `https://www.iso.org/standard/74947.html`
- ステークホルダーコミュニケーションと品質確保を行う

### ISO 10002:2018
- 参照: `https://www.iso.org/standard/71580.html`
- 苦情受付・分類・原因分析・是正・再発防止の運用を整備する

### ISO 10004:2018
- 参照: `https://www.iso.org/standard/71582.html`
- 顧客満足の指標設計と測定・レビューを行う

### ISO 30401:2018
- 参照: `https://www.iso.org/standard/68683.html`
- 知識の特定・作成・共有・維持のプロセスと責任を定義する

### IEC 31010:2019
- 参照: `https://www.iso.org/standard/72140.html`
- リスク評価技法を選定し、対象事象に応じて使い分ける

### ISO 10001:2018
- 参照: `https://www.iso.org/standard/71579.html`
- 顧客対応の行動規範（約束）を策定し、周知・逸脱対応を整備する

### ISO 18295-1:2017
- 参照: `https://www.iso.org/standard/64739.html`
- コンタクトセンターの運用要件（応対品質/教育/モニタリング）を整備する

### ISO 9004:2018
- 参照: `https://www.iso.org/standard/70397.html`
- 組織の持続的成功に向け、成熟度自己評価と改善を運用する
