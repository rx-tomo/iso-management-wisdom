# プロジェクトマネージャ（PM）（Role Skill）

## このロールの専門性
- スコープ/スケジュール/コスト/品質の統合管理
- リスク/課題/変更の運用（合意形成）
- 透明な報告（ステータス/見通し/意思決定）

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 21502:2020
- 参照: `https://www.iso.org/standard/74947.html`
- 目的・成果物・制約を明確化し、計画を策定する
- 進捗・課題・変更・リスクを運用し、意思決定を記録する
- ステークホルダーコミュニケーションと品質確保を行う

### ISO 22301:2019
- 参照: `https://www.iso.org/standard/75106.html`
- BCPと復旧手順、訓練/演習を計画・実施する

### ISO 21500:2021
- 参照: `https://www.iso.org/standard/75704.html`
- プロジェクト/プログラム/ポートフォリオの文脈と概念を統一し、適用範囲を定義する

### ISO 21506:2024
- 参照: `https://www.iso.org/standard/87900.html`
- PM用語を定義し、文書・会議体・報告で一貫性を保つ

### ISO 21508:2018
- 参照: `https://www.iso.org/standard/63582.html`
- 出来高管理（EVM）の基準線と指標を定義し、レポーティングを運用する

### ISO 21511:2018
- 参照: `https://www.iso.org/standard/69702.html`
- WBSを用いて作業分解・責任分担・見積/計画の基盤を整備する

### ISO 21512:2024
- 参照: `https://www.iso.org/standard/63584.html`
- EVM導入手順（役割/頻度/レポート）を定義し、定着を支援する

### ISO 14064-2:2019
- 参照: `https://www.iso.org/standard/66454.html`
- プロジェクト単位のGHG削減量/吸収量の算定・監視・報告を整備する
