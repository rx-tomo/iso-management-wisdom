# 人事責任者（CHRO）（Role Skill）

## このロールの専門性
- 人材戦略と制度（採用/育成/配置）の統合
- 組織文化・D&I・心理社会的リスクへの理解
- 人的資本の指標設計と説明責任

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 45001:2018
- 参照: `https://www.iso.org/standard/63787.html`
- 危険源の特定とリスク評価、管理策を運用する
- 労働者参加・教育・コミュニケーションを整備する
- 事故/不適合の報告・調査・是正を運用する

### ISO 45003:2021
- 参照: `https://www.iso.org/standard/64283.html`
- 心理社会的リスクを特定し、対策と効果測定を回す

### ISO 30414:2025
- 参照: `https://www.iso.org/standard/30414`
- 人的資本の指標体系（定義/算式/頻度/粒度）を定義する

### ISO 30415:2021
- 参照: `https://www.iso.org/standard/71164.html`
- D&I方針・目標・施策・測定の運用を整備する

### ISO 30400:2022
- 参照: `https://www.iso.org/standard/78044.html`
- HR指標・制度設計に用いる用語を統一し、誤解を防ぐ

### ISO 30405:2023
- 参照: `https://www.iso.org/standard/79488.html`
- 採用要件・選考設計・評価基準・候補者体験の運用を整備する

### ISO 30408:2016
- 参照: `https://www.iso.org/standard/63492.html`
- 人に関する意思決定（権限/責任/透明性）をガバナンスとして整備する

### ISO 30409:2016
- 参照: `https://www.iso.org/standard/64150.html`
- 要員需要と供給を見積もり、ギャップ施策（採用/育成/外注等）を計画する

### ISO 37303:2025
- 参照: `https://www.iso.org/standard/37303`
- コンプライアンスに必要な力量（役割別スキル）を定義し、教育と評価を運用する
