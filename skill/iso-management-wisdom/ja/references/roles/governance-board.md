# 取締役会/監督機関（Role Skill）

## このロールの専門性
- ガバナンス（説明責任/監督/牽制）の原則理解
- リスクと機会を経営判断へ統合できる
- 独立性・透明性・長期視点

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 37000:2021
- 参照: `https://www.iso.org/standard/65036.html`
- ガバナンス原則・価値創出/保護の方針を定義する
- 役割責任（監督/執行/牽制）と意思決定ルールを定義する
- ガバナンスの有効性を定期レビューし、改善を指示する

### ISO/IEC 38500:2024
- 参照: `https://www.iso.org/standard/81684.html`
- IT投資・意思決定の原則と責任分界を定義する

### ISO 21505:2017
- 参照: `https://www.iso.org/standard/63578.html`
- プロジェクト/プログラム/ポートフォリオのガバナンス（責任/監督/報告）を定義する

### ISO 30408:2016
- 参照: `https://www.iso.org/standard/63492.html`
- 人に関する意思決定（権限/責任/透明性）をガバナンスとして整備する

### ISO/IEC 27014:2020
- 参照: `https://www.iso.org/standard/74046.html`
- 情報セキュリティのガバナンス（経営報告、責任分界、監督）を整備する
