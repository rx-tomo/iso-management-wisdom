# 代表取締役/CEO（経営責任者）（Role Skill）

## このロールの専門性
- 方針と優先順位を決め、資源配分できる
- 重要リスクの受容/回避/軽減の判断ができる
- 仕組みを"回す"執行力（会議体/レビュー）

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 37000:2021
- 参照: `https://www.iso.org/standard/65036.html`
- ガバナンス原則・価値創出/保護の方針を定義する

### ISO 31000:2018
- 参照: `https://www.iso.org/standard/65694.html`
- リスクマネジメント方針・原則・枠組みを定義する

### ISO 37001:2025
- 参照: `https://www.iso.org/standard/37001`
- 贈収賄防止方針と責任者体制を定義する

### ISO/IEC 38500:2024
- 参照: `https://www.iso.org/standard/81684.html`
- IT投資・意思決定の原則と責任分界を定義する

### ISO/IEC 27001:2022
- 参照: `https://www.iso.org/standard/82875.html`
- 内部レビューとマネジメントレビューを実施する

### ISO/IEC 42001:2023
- 参照: `https://www.iso.org/standard/42001`
- AI利用の方針・目的・責任分界を定義し、経営として承認する

### ISO 21504:2022
- 参照: `https://www.iso.org/standard/82867.html`
- 投資配分・優先順位・中止判断を行い、レビュー会議体を運用する

### ISO 26000:2010
- 参照: `https://www.iso.org/standard/42546.html`
- 社会的責任（SR）の重要課題（マテリアリティ）と方針を整理する

### ISO 56001:2024
- 参照: `https://www.iso.org/standard/79278.html`
- イノベーション方針・目標・役割責任を定義する

### ISO/UNDP DIS 53001
- 参照: `https://www.iso.org/standard/53001`
- SDGsをマネジメントシステムとして運用（方針/目標/評価/改善）する枠組みを整備する
