# サステナビリティ/ESG責任者（Role Skill）

## このロールの専門性
- ESGの方針→目標→実行→評価の運用設計
- GHG算定/主張/検証の要求理解
- 調達・投融資・事業運営への統合

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 20400:2017
- 参照: `https://www.iso.org/standard/63026.html`
- サステナブル調達方針と評価基準（ESG含む）を定義する

### ISO 26000:2010
- 参照: `https://www.iso.org/standard/42546.html`
- 社会的責任（SR）の重要課題（マテリアリティ）と方針を整理する
- ステークホルダー関与と説明責任（開示/対話）を運用する

### ISO 14064-1:2018
- 参照: `https://www.iso.org/standard/66453.html`
- 組織のGHG算定境界・算定ルール・証跡・報告を定義する

### ISO 14068-1:2023
- 参照: `https://www.iso.org/standard/43279.html`
- カーボンニュートラル/ネットゼロ主張の条件と証跡を整備する

### ISO 37101:2016
- 参照: `https://www.iso.org/standard/61885.html`
- サステナビリティの方針・目標・運用・評価・改善をMSとして整備する

### ISO 14064-2:2019
- 参照: `https://www.iso.org/standard/66454.html`
- プロジェクト単位のGHG削減量/吸収量の算定・監視・報告を整備する

### ISO 14064-3:2019
- 参照: `https://www.iso.org/standard/66455.html`
- GHG声明の検証/妥当性確認に備え、証跡と是正プロセスを整備する

### ISO 14067:2018
- 参照: `https://www.iso.org/standard/71206.html`
- 製品カーボンフットプリント（CFP）の算定境界・データ要件・報告を整備する

### ISO 14083:2023
- 参照: `https://www.iso.org/standard/78864.html`
- 輸送連鎖のGHG排出量算定と報告（データ要件/境界）を整備する

### ISO 14090:2019
- 参照: `https://www.iso.org/standard/68507.html`
- 気候変動適応の原則・要求事項に基づく運用（計画/統制）を整備する

### ISO 14091:2021
- 参照: `https://www.iso.org/standard/68508.html`
- 脆弱性・影響・リスク評価の手順とレビュー運用を整備する

### ISO/TS 14092:2020
- 参照: `https://www.iso.org/standard/68509.html`
- 地域/コミュニティの適応計画の要求と手順を整備する

### ISO 14093:2022
- 参照: `https://www.iso.org/standard/68511.html`
- 適応資金（助成等）の要求と評価の枠組みを整備する

### ISO 14097:2021
- 参照: `https://www.iso.org/standard/72433.html`
- 投融資活動に関する気候関連の評価と報告の枠組みを整備する

### ISO/UNDP DIS 53001
- 参照: `https://www.iso.org/standard/53001`
- SDGsをマネジメントシステムとして運用（方針/目標/評価/改善）する枠組みを整備する
