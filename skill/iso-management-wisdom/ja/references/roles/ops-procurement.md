# 調達責任者（Role Skill）

## このロールの専門性
- サプライヤ選定/評価/契約要求の設計
- 供給網リスク（ESG/セキュリティ/品質）管理
- 外部委託の監督（SLA/監査/是正）

## 手順（チェックリスト）
- 目的とスコープ（対象組織/対象業務/対象期間）を最初に固定する。
- 以下の規格別チェックを行い、必要な成果物/意思決定/レビューを洗い出す。

### ISO 37001:2025
- 参照: `https://www.iso.org/standard/37001`
- 第三者デューデリジェンスと取引管理（贈答/接待等）を運用する

### ISO 20400:2017
- 参照: `https://www.iso.org/standard/63026.html`
- サステナブル調達方針と評価基準（ESG含む）を定義する
- サプライヤの評価・改善（監査/是正）を運用する

### ISO 18295-2:2017
- 参照: `https://www.iso.org/standard/64740.html`
- 委託先コンタクトセンターの要求・評価・責任分界（SLA等）を定義する

### ISO/IEC 27017:2015
- 参照: `https://www.iso.org/standard/43757.html`
- クラウドの共有責任モデルを明確化し、統制を具体化する

### ISO/IEC 27036-2:2022
- 参照: `https://www.iso.org/standard/82060.html`
- 供給者関係の要求事項（評価/契約/監査/是正）を整備する

### ISO/IEC 27036-3:2023
- 参照: `https://www.iso.org/standard/82890.html`
- サプライチェーンセキュリティ（HW/SW/サービス）の評価と管理を運用する

### ISO 14067:2018
- 参照: `https://www.iso.org/standard/71206.html`
- 製品カーボンフットプリント（CFP）の算定境界・データ要件・報告を整備する

### ISO 14083:2023
- 参照: `https://www.iso.org/standard/78864.html`
- 輸送連鎖のGHG排出量算定と報告（データ要件/境界）を整備する
